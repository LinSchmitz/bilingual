## ساختار نهایی پوشه‌ها و فایل‌های پروژه‌ی خورشیدبان

```
خورشیدبان/
├── public/
│ └── index.html

├── src/
│
│ ├── assets/ # تصاویر، آیکون‌ها و فایل‌های ثابت
│
│ ├── components/ # کامپوننت‌های قابل استفاده مجدد
│ │ ├── Navbar/ # منوی بالایی شامل لوگو، انتخاب تقویم، زبان و جستجو
│ │ ├── Sidebar/ # فیلترها (نوع تقویم، دسته‌بندی، بازه زمانی)
│ │ ├── TodayCard/ # کارت نمایش "امروز چه روزی‌ست؟"
│ │ ├── EventBox/ # باکس هر مناسبت جداگانه
│ │ ├── EventList/ # لیست مناسبت‌های امروز و هفته
│ │ ├── Calendar/ # تقویم کامل
│ │ ├── Zodiac/ # نمایش برج فلکی بر اساس تاریخ
│ │ └── Footer/ # راه ارتباطی، منبع داده‌ها، و تیم
│
│ ├── layouts/ # چیدمان کلی صفحات (مثلاً MainLayout.jsx)
│
│ ├── pages/ # صفحات اصلی مثل Home، About، Events
│ │ ├── HomePage.jsx
│ │ ├── EventsPage.jsx
│ │ └── AboutPage.jsx
│
│ ├── services/ # ارتباط با Supabase، API یا دیتابیس
│ │ └── supabase.js
│
│ ├── utils/ # توابع کمکی (مثل محاسبه تاریخ‌ها، برج‌ها)
│ │ ├── dateHelpers.js
│ │ ├── zodiacUtils.js
│ │ └── calendarConverters.js
│
│ ├── constants/ # لیست تقویم‌ها، دسته‌بندی‌ها، رنگ‌ها، مناسبت‌های استاتیک
│ │ └── calendarTypes.js
│
│ ├── styles/ # فایل‌های CSS یا SCSS
│ │ ├── globals.css # استایل‌های کلی پروژه
│ │ └── variables.css # رنگ‌ها و متغیرهای CSS
│
│ ├── App.jsx # فایل اصلی App
│ └── index.js # ورود به React DOM

```

## 🧱 کامپوننت‌های React

```
| کامپوننت        | توضیح                                                   |
| --------------- | -------------------------------------------------------- |
| `Navbar`        | انتخاب نوع تقویم، زبان، سرچ                            |
| `Sidebar`       | فیلتر بر اساس تقویم یا نوع مناسبت                      |
| `TodayCard`     | نمایش تاریخ امروز (شمسی، پهلوی، ...)، ساعت، برج فلکی  |
| `EventBox`      | نمایش هر مناسبت به‌صورت جداگانه                         |
| `EventList`     | لیست مناسبت‌های امروز و هفته                            |
| `CalendarView`  | تقویم کامل با مناسبت‌ها                                  |
| `ZodiacDisplay` | نمایش برج فلکی و مشخصات                                 |
| `Footer`        | اطلاعات سایت، تماس، منابع داده                           |
```

## 🧠 ۳. فایل‌های منطقی (Logic)

### ✅ `utils/zodiac.js`

تابع `getZodiacSign(month, day)` + دیتا برج‌ها

---

### 🗓️ `utils/convertDate.js`

تبدیل تاریخ شمسی ↔ میلادی ↔ زرتشتی (با استفاده از کتابخانه‌هایی مثل `moment-jalaali`, `jalaali-js`, یا خودت)

---

### 📡 `services/supabase.js`

اتصال و توابع CRUD برای دریافت مناسبت‌ها از Supabase

```js
import { createClient } from '@supabase/supabase-js';

const supabase = createClient(YOUR_SUPABASE_URL, YOUR_SUPABASE_KEY);
export default supabase;
```

## 🎁 نمونه‌های پیشرفته (بعداً اضافه کن)

- انتخاب تاریخ دلخواه با نمایش اطلاعات آن روز
- تبدیل تاریخ تولد کاربر به سبک زرتشتی یا شاهنشاهی
- نوتیفیکیشن برای مناسبت‌های خاص
- API عمومی برای نمایش مناسبت‌های ایرانی در دیگر وب‌سایت‌ها
- نمایش زمان واقعی سروری
- تقویم طولانی (۳۰۰۰ سال) و تبدیل تاریخ
- لیست مناسبت‌های رسمی
- جمله روزانه یا معرفی کوتاه
- داده‌ها را از Supabase بخوانیم
- فیلتر و جستجو را اضافه کنیم
- بخش تقویم کامل را بسازیم (هفتگی/ماهانه)
- آماده‌سازی نسخه موبایل

## 📱 حالت موبایل

در موبایل:

- نوار بالا جمع می‌شود و با آیکون همبرگری (`☰`) باز می‌شود.
- مناسبت‌های امروز و دکمه‌ها در یک ستون زیر هم قرار می‌گیرند.
- تاریخ‌ها و مناسبت‌ها در قالب کارت‌های عمودی لیست می‌شوند.
